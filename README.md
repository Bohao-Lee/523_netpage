# 523班 · 高中毕业十周年纪念网站 🎓

> 2016 - 2026 · 十年光阴，各自精彩

## 项目简介

这是一个用于记录523班同学毕业十年后工作情况的纪念网站。网站以中国地图为主体，展示各位同学在全国各省份的分布情况，点击各省份可查看该地区同学的详细工作信息。

## 功能特点

- 🗺️ **交互式中国地图** - 直观展示同学地理分布
- 🎨 **颜色标识** - 根据人数多少用不同颜色标记省份
- 📍 **点击查看详情** - 点击省份查看该地区所有同学信息
- 🔍 **搜索筛选** - 支持按姓名、省份、行业筛选
- 📱 **响应式设计** - 支持手机、平板、电脑多端访问

## 如何添加/修改同学信息

编辑 `js/classmates-data.js` 文件，按照以下格式添加同学信息：

```javascript
{
    name: "张三",           // 姓名
    province: "北京",       // 省份（必须与地图省份名称一致）
    city: "北京市",         // 城市
    district: "海淀区",     // 区县
    company: "字节跳动",    // 公司名称
    position: "产品经理",   // 职位
    industry: "互联网",     // 行业
    message: "期待相聚！",  // 寄语（可选）
    avatar: ""              // 头像URL（可选）
}
```

## 如何运行

### 方法一：直接打开
双击 `index.html` 文件在浏览器中打开。

### 方法二：使用本地服务器（推荐）

```bash
# 使用 Python 3
python -m http.server 8080

# 使用 Node.js (需要安装 http-server)
npx http-server -p 8080

# 使用 VS Code Live Server 插件
# 右键 index.html -> Open with Live Server
```

然后访问 `http://localhost:8080`

## 自定义

### 修改主题颜色

编辑 `css/style.css` 中的 CSS 变量：

```css
:root {
    --primary-color: #667eea;      /* 主色调 */
    --secondary-color: #764ba2;    /* 辅助色 */
    --accent-color: #f093fb;       /* 强调色 */
}
```

### 修改标题

编辑 `index.html` 中的标题部分：

```html
<h1>🎓 523班 · 高中毕业十周年纪念</h1>
<p class="subtitle">2016 - 2026 · 十年光阴，各自精彩</p>
```
